Tecnologias que eu utilizo

HTML 5

Html é uma linguagem de marcação ela é a responsável por adicionar o conteúdo ao site.

CSS 3

Css é outra linguagem de marcação, ela está trabalhando em conjunto com o html para criar este site, o CSS é o responsável por adicionar o estilo ao meu site.

JS

Javascript é a linguagem de programação que é responsável pelo back-end atráves dessa ferramenta é possível criar sites dinâmicos, juntos com o html e css nós temos um dos principais conjuntos de ferramentas na criação de um site.

Python

O python é uma linguagem de programação super prática e fácil de aprender e graças a sua quantidade de bibliotecas ela acaba se tornando bastante flexível.
